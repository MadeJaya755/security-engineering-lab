# Day 02 - Password Hashing with bcrypt

## Problem
Menyimpan password dalam bentuk plain text berisiko tinggi jika terjadi data breach.

## Solution
Menggunakan bcrypt untuk hashing password dengan salt otomatis.

## Use Case
Digunakan dalam sistem login, authentication service, dan user management.

## Key Features
- Secure hashing (bcrypt)
- Password verification
- Salt otomatis

## Limitation
- Tidak cocok untuk data non-password
