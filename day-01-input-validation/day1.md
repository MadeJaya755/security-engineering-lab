# Day 01 - Input Validation Library

## Problem
User input dapat mengandung payload berbahaya seperti SQL Injection dan XSS.

## Solution
Membuat validasi dan sanitasi input menggunakan regex sederhana.

## Use Case
Digunakan pada form login, search bar, dan API input handling.

## Limitation
- Regex masih sederhana
- Tidak menggantikan ORM atau security framework
